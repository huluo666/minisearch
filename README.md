# minisearch
Mac基于fd命令行的GUI应用

https://github.com/sharkdp/fd



#### fd安装

```sh
$ brew install fd
```



> An electron-vue project

#### Build Setup

```
# install dependencies
npm install

# serve with hot reload at localhost:9080
npm run dev

# build electron application for production
npm run build
```
